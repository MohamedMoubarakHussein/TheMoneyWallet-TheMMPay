
A user service:
    As a guest, I want to register for an account so that I can access the app.
    As a user, I want a secure Authentication process so that I can log in and manage my account.     
    As a user, I want to validate my email address and phone number.
    As a user, I want to set up and manage my profile to provide my information and keep my data up to date.
    As an admin, I want to create special accounts for manager so that they can see usage detailes of the app.
    

 

A wallet service:
    As a user, I want to create a digital wallet and linked to my account so that I can manage my funds.
    As a user, I want to easily check my wallet balance so that I can monitor my funds.

A transaction service:
    As a user, I want to add funds to my wallet using various methods.
    As a user, I want to withdraw funds to my wallet using various methods.
    As a user, I want to send send money to other users using their usernames or phone numbers.
    As a user, I want to receive money from other users.
    As a user, I want to pay my bills directly from the app.


A history service:
    As a user, I want to view a detailed transaction history, including dates and amounts.

A notification service:
    As a user, I want to receive a real time updates on my wallet after each transaction. 
    As a user, I want to receive  instant notification when someone sends money to my wallet.
    As a user, i want to receive notification for every transaction.
